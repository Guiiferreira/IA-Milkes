## 🧾 Atividades do Repositório

### 📌 Atividade 1 — Análise Exploratória de Dados

A primeira atividade presente no repositório é uma **Análise Exploratória de Dados**, que consiste em:

- Explorar um conjunto de dados  
- Investigar e entender suas características principais  
- Realizar estatísticas descritivas  
- Produzir visualizações para facilitar a interpretação dos dados


### 📌 Atividade 2 — Conclusão da Análise Exploratória de Dados

A segunda atividade consiste na **execução do notebook `IA Lab-Aula 2.ipynb`** e na elaboração de uma conclusão detalhada baseada na Análise Exploratória de Dados realizada.

O objetivo da atividade é:

- Executar o notebook disponibilizado pela professora  
- Interpretar os resultados obtidos durante a análise exploratória  
- Identificar padrões, tendências e possíveis relações entre os dados  
- Elaborar uma conclusão detalhada e profissional destacando os principais achados da análise

Esta análise foi desenvolvida como parte das atividades avaliativas propostas pela professora Prof. Milkes Yone Alvarenga.
